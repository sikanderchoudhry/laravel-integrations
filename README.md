## About Laravel Integrations

This project aims to provide integrations to common apis in laravel. Current list of integrations includes:
1. Mailchimp
2. Instgram
3. Twilio
4. Click Funnels
5. Slack
6. Facebook
7. Zapier
8. Drip
9. ConvertKit
10. SendGrid
11. AWeber
12. ActiveCampaign
13. Infusionsoft
14. Klaviyo
